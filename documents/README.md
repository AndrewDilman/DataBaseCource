# Проект Маркетплейс - Обзор

## Введение

Данный проект представляет собой учебную платформу маркетплейса с полноценной Multi-DB архитектурой. Проект демонстрирует работу с различными типами баз данных, их интеграцию, а также паттерны проектирования распределённых систем.

## Технологический стек

| Компонент | Версия | Назначение |
|----------|--------|-----------|
| PostgreSQL | 15 | основная реляционная БД |
| PostgreSQL (subscriber) | 15 | реплика для чтения |
| MongoDB | 6 | документная БД для каталога |
| Redis | 7-alpine | кэширование |
| Neo4j | 5 | графовая БД |
| Apache Kafka | 3.7 | событийный стриминг |
| Prometheus | 2.40 | сбор метрик |
| Grafana | enterprise | визуализация |
| Node.js API | Express | REST сервис |

## Компоненты баз данных

### PostgreSQL (primary)
Основная транзакционная база данных marketplace содержит:
- Пользователи (покупатели и продавцы)
- Товары и категории
- Заказы и история покупок
- Отзывы и рейтинги
- Адреса, пункты выдачи, склады

### PostgreSQL (subscriber)
Read replica для:
- Горизонтального масштабирования чтения
- Отказоустойчивости
- Аналитических запросов

### MongoDB
Документная база для:
- Каталога товаров (гибкая схема)
- Отзывов с валидацией
- Временных событий (TTL)

### Redis
Кэширующий слой:
- Кэширование тяжёлых агрегаций
- Session management
- Rate limiting

### Neo4j
Графовая база для:
- Анализа связей между сущностями
- Рекомендаций
- Социального графа

### Kafka
Событийный стриминг (kafka-lab):
- Паттерн Outbox
- CDC (Change Data Capture)
- Агрегации событий через Streams

## Мониторинг

- **Prometheus** собирает метрики со всех компонентов
- **Grafana** предоставляет дашборды
- **postgres-exporter** - метрики PostgreSQL
- **mongodb-exporter** - метрики MongoDB
