MATCH (c:Category)<-[:IN_CATEGORY]-(p:Product)-[:SOLD_BY]->(m:User)
WHERE m.type = 'merchant'
WITH c, m, count(p) as productCount
ORDER BY c.categoryId, productCount DESC
WITH c, collect({merchant: m.name, products: productCount, id: m.userId}) as merchants
RETURN c.categoryId as catId,
       c.name as category, 
       [x IN merchants[0..3] | x.merchant][0..3] as top3_merchants,
       [x IN merchants[0..3] | x.products][0..3] as product_counts
ORDER BY catId

Как работает:

MATCH (c:Category)<-[:IN_CATEGORY]-(p:Product)-[:SOLD_BY]->(m:User) — связываем категорию с товаром и продавцом
WHERE m.type = 'merchant' — фильтруем только продавцов (не покупателей)
count(p) as productCount — считаем количество товаров у каждого продавца в категории
collect() — собираем всех продавцов категории в список, отсортированный по убыванию
[merchants[0..3]] — берём только первые 3 элемента (топ-3)
Результат: таблица с колонками:

catId — ID категории
category — название категории
top3_merchants — имена топ-3 продавцов
product_counts — количество товаров каждого из них
